# lp-fix-demo（スマホ崩れ修正デモ）

## 公開URL
https://s-iz-dev.github.io/lp-fix-demo/

## 概要
スマホ表示でよくある「表示崩れ」を，Before／Afterの切替で確認できるデモページです．
実案件の修正対応ができることを，短時間で伝える目的で作成しました．

## デモ内容
- 横スクロールが出る（固定幅が原因）→ 画面幅に収めて解消
- 画像がはみ出す（固定サイズが原因）→ 親要素に追従して解消
- ボタンが押しにくい（タップ領域が小さい）→ 最小44px目安で改善

## できること
- スマホ表示の崩れ修正（レスポンシブ調整）
- CSS調整（flex/grid，余白，幅，高さ）
- 画像表示の調整（max-width，object-fit など）
- 軽微なJavaScript追加（UI切替，簡単な動作）

## 対応外
- ログイン機能，決済機能，会員登録
- 個人情報DBを扱う実装
- 大量スクレイピング
